[1] 로비 UI 구상

--> 배경 ( 스크롤 기능, 용병들로 부터 퀘스트 접수, 특수 상인 등장 )
--> 인벤토리 ( 무기, 방어구, 강화 재료, 아이템 4가지를 담은 공간 구현 ) 
--> 강화 ( 무기 및 방어구의 성능을 강화하는 공간 구현 )
--> 상점 ( 무기, 방어구를 구입 할 수 있는 공간 및 강화 재료, 아이템을 팔 수 있는 공간 구현 )
[ 단, 상점에서는 강화 재료 및 아이템을 구매 할 수 없다. ]

강화 재료는 용병 퀘스트 및 특수 상인의 거래, 확률적으로 좀비들에게서 얻을 수 있다.
강화 재료는 무기 와 방어구에 결합하여 성능을 최대화 할 수 있다. 
( 하나의 장비당 최대 3개까지 결합하여 성능을 증가시킬 수 있다. )
또한, 용병들의 무기에도 강화 재료를 쓸 수 있다.



[2] 아이템 계열 CLASS 구현

Item 클래스를 구현한다.
Item 클래스에서는 총 3가지 타입이 있으며, 방어구, 강화 재료, 아이템을 구성한다.
각 좀비들은 다양한 강화 재료, 아이템 두 가지 타입만 가진다.
( 무기와 방어구는 무조건 상점에서 구매할 수 있다. )

Item은 노멀, 레어, 에픽, 유니크, 레전드리 총 5가지의 등급이 매겨진다.
등급에 따라 아이템의 효과는 천지차별이다.

Item은 용병 퀘스트, 특수 상인, 좀비들에게서 얻을 수 있다.

<아이템 별 색상>
노멀 - 흰색
레어 - 하늘색
에픽 - 보라색
유니크 - 노란색
레전드리 - 빨간색
